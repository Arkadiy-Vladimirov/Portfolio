В этой папке находятся: 
-скрипт программы s_task01. Его нужно запустить для демонстрации работы программы (предварительно запустить скрипт s_config!);
-файл функции lpsp (linear processing speed problem) которая решает заданную посредством входных параметров линейную задачу быстродействия;
-скрипт s_config. s_config задает параметры задачи быстродействия и сохраняет их в .mat файл, хранящийся в папке configs;
-функия rhoEll, аналитически считающая опорную функцию эллипса, чем и являются в данном варианте множество начальных точек и множество допустимых значений управления;

-папка configs(см. пред. пункт);
-папка examples с сохраненными графиками некоторых решений (нужны в первую очередь для отчета);
-.tex исходник отчета -- файл No1;
-файл отчета No1.pdf;

прочие вспомогательные файлы.
